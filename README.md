# 💸 Python Expense Tracker

A command-line expense tracking application built with Python.  
This project allows users to record, analyze, and manage their expenses with a clean and interactive CLI interface.

---

## 🚀 Features

- ➕ Add expenses with amount, category, and date
- 📋 View all expenses (sorted by highest amount)
- 💰 Calculate total spending
- 🔍 Filter expenses by category
- 📊 Summary of expenses by category
- ✅ Input validation (prevents invalid entries)
- 🧠 Smart user input handling (accepts flexible commands like "1", "add", etc.)

---

## 🛠️ Technologies Used

- Python
- CSV (file handling)
- Functions & modular structure
- Loops & conditionals
- `map()` and `filter()`
- Error handling (`try/except`)
- Sorting with `lambda`
